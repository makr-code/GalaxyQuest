# Die Ketzer von Verath – Abtrünnige Syl'Nar

## Überblick

Die Ketzer von Verath sind eine dezentralisierte Gemeinschaft abtrünniger Syl'Nar, gegründet von einem exkommunizierten Priester, der behauptete, die Leerenbrut sei keine externe Invasion, sondern eine Konsequenz des kosmischen Ungleichgewichts, das die Konvergenz selbst verursacht. Versteckt in einem verborgenen Kloster-Riff auf dem Planeten Verath, sammeln sie verbotene Texte und verbotenes Wissen. Sie sind keine Terroristen, sondern Wahrheitssuchende – aber ihre Wahrheiten sind so explosiv, dass mächtige Institutionen sie vernichten wollen. Sie besitzen möglicherweise den Schlüssel zur Leerenbrut-Lösung.

## Zeitleiste

| Zeitpunkt | Ereignis |
|-----------|----------|
| Vor ~2.000 Jahren | Priester Verath entdeckt Texte über Leerenbrut als Konsequenz des Ungleichgewichts |
| Vor ~1.800 Jahren | Verath exkommuniziert; Anhänger gründen verstecktes Kloster-Riff |
| Vor ~1.200 Jahren | Erstes Ketzer-Konzil; Sammlung verbotener Texte aller Spezies |
| Vor ~400 Jahren | Friedliche Phase; Ketzer als neutrale Vermittler in Grenzkonflikten aktiv |
| Vor ~80 Jahren | Entdeckung einer Aethernox-Datendisk; Syl'Nar-Rat schickt Agenten zur Vernichtung |
| Gegenwart | Ketzer versuchen Beweise dem Konvergenz-Rat zu präsentieren |

## 3-Akt Plot-Outline

### Akt 1 – Die Verfolgten
Spieler begegnen verfolgten Ketzer-Pilgern. Schutz gewähren führt zum Verath-Kloster und Sael-Voraths Einladung.

### Akt 2 – Die Datendisk
Die Datendisk ist gefunden aber unvollständig. Thel-Nessa braucht einen Aethernox-Monolithen um sie zu lesen. Syl'Nar-Agenten und Leth-Ryahs Verteidigung kollidieren.

### Akt 3 – Explosive Wahrheit
Der vollständige Text beweist: Vela'Thii wusste von der Leerenbrut-Ursache und schwieg. Die Spieler entscheiden wie sie dieses Wissen einsetzen und wen sie schützen.

### Nachbeben (Wiederholbar)
Neue verbotene Texte werden entdeckt; Syl'Nar-Agenten infiltrieren Spieler-Verbündete; das Kloster-Riff wird von der Leerenbrut bedroht.
