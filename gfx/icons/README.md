# SVG Icon Sets

UI-Icons fuer GalaxyQuest als einzelne Dateien pro Icon.

## Struktur

- `mono/` enthaelt das monochrome Set
- `duotone/` enthaelt das zweifarbige Set
- `colorful/` enthaelt das bunte Set

Alle Sets enthalten pro Icon genau eine Datei, z. B. `home.svg`.

## Nutzung

```html
<svg class="gq-icon gq-icon-btn" aria-hidden="true" focusable="false">
  <use href="gfx/icons/mono/home.svg#icon-home"></use>
</svg>

<svg class="gq-icon gq-icon-btn" aria-hidden="true" focusable="false">
  <use href="gfx/icons/duotone/home.svg#icon-home"></use>
</svg>

<svg class="gq-icon gq-icon-btn" aria-hidden="true" focusable="false">
  <use href="gfx/icons/colorful/home.svg#icon-home"></use>
</svg>
```

Fuer zugaengliche Buttons immer einen `aria-label` setzen.
